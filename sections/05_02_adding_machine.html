10. Ứng dụng: Cấu trúc Adding Machine

System-Level Scan Design

  • Adding Machine: Một hệ thống cấp cao chuẩn (RTL) bao gồm Data Path (ALU, MUX, Registers) và Control Unit (FSM). Chứa hàng trăm đường tín hiệu.
  • Phân vùng (Partitioning): Chia mạch thành 2 chuỗi Scan độc lập (Ví dụ: Chuỗi 1 cho Control Unit, Chuỗi 2 cho các thanh ghi Data Path).
  • Virtual Tester ATE: Máy test có thể thao tác đồng thời bơm tín hiệu độc lập vào Control Unit và Data Path, giúp test coverage tiệm cận mức hoàn hảo ở mức Data Level.
